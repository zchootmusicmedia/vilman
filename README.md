# המכתב שלא הצלחתי לכתוב לך

## העלאה ל-GitHub Pages
1. חלצי את קובץ ה-ZIP.
2. העלי את כל הקבצים והתיקיות לשורש המאגר `vilman`.
3. ודאי ש-GitHub Pages מופעל.
4. עמוד ההרשמה: `https://zchootmusicmedia.github.io/vilman/`
5. דף התודה / המשחק: `https://zchootmusicmedia.github.io/vilman/game.html`

## חשוב
- בטופס רב-מסר יש להגדיר את `game.html` כדף תודה.
- בחירת ילד/ילדה נשמרת בדפדפן וממשיכה לדף המשחק.
- הורדת PDF משתמשת בספריות חיצוניות של html2canvas ו-jsPDF ולכן דורשת חיבור לאינטרנט.


גרסה זו כוללת תמונות WebP מכווצות ומותאמת להעלאה קלה ל-GitHub Pages.
